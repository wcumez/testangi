# Testangi

Este proyecto es una página web sencilla de tipo landing page creada con HTML y CSS. La página presenta una interfaz moderna, limpia y visualmente atractiva, con un mensaje principal de bienvenida y un llamado a la acción para explorar los servicios ofrecidos.

## ¿Qué muestra la página?

- Un encabezado con navegación básica.
- Un diseño con estilo moderno y fondo degradado.
- Un mensaje principal que invita al usuario a conocer la propuesta.
- Un botón de acción para comenzar.

## Tecnologías utilizadas

- HTML
- CSS

## Objetivo

Servir como una portada simple y elegante para presentar una marca o proyecto en línea.
